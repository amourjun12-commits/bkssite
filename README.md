# BKS Construction Landing Page

A premium, modern landing page for BKS Construction - Benin's leading construction material delivery service.

## Project Structure
- `index.html`: Main landing page structure
- `styles.css`: Custom CSS with dark mode industrial theme
- `script.js`: Interactive elements (mobile menu, scroll effects)
- `assets/`: Premium generated imagery

## Deployment

### One-Click Deploy to Vercel
Click the button below to deploy this project to Vercel instantly:

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/amourjun12-commits/bkssite)

### Manual Deployment
1. Clone the repository
2. Open `index.html` in your browser
3. Or deploy via Vercel CLI: `vercel`
